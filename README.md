# Calculadora Python

Este projeto é uma calculadora simples escrita em Python que permite realizar operações aritméticas básicas como adição, subtração, multiplicação e divisão.

## Funcionalidades

- Adição (`+`)
- Subtração (`-`)
- Multiplicação (`*`)
- Divisão (`/`)
